# 安全与可靠性检查清单

## 输入/输出安全

- **XSS**：不安全的 HTML 注入、`dangerouslySetInnerHTML`、未转义的模板、innerHTML 赋值
- **注入攻击**：通过字符串拼接或模板字面量的 SQL/NoSQL/命令/GraphQL 注入
- **SSRF**：用户可控的 URL 未经白名单校验即访问内部服务
- **路径遍历**：用户输入用于文件路径且未经过滤（`../` 攻击）
- **原型污染**：JavaScript 中不安全的对象合并（`Object.assign`、展开运算符处理用户输入）

## AuthN/AuthZ

- 读写操作缺失租户或所有权检查
- 新增端点未设置认证守卫或 RBAC 控制
- 信任客户端提供的角色/标志/ID
- 越权访问（IDOR - Insecure Direct Object Reference）
- Session 固定攻击或弱会话管理

## JWT 与 Token 安全

- 算法混淆攻击（期望 `RS256` 时接受了 `none` 或 `HS256`）
- 弱密钥或硬编码密钥
- 缺失过期时间（`exp`）或未校验过期
- JWT payload 中包含敏感数据（Token 是 base64 编码，非加密）
- 未校验 `iss`（签发者）或 `aud`（受众）

## 密钥与 PII

- API Key、Token 或凭证出现在代码/配置/日志中
- 密钥存在于 git 历史或暴露给客户端的环境变量中
- 过度记录 PII 或敏感数据
- 错误消息中缺失数据脱敏

## 供应链与依赖

- 未锁定的依赖版本允许恶意更新
- 依赖混淆攻击（私有包名冲突）
- 从不可信来源或 CDN 导入且无完整性校验
- 存在已知 CVE 的过时依赖

## CORS 与请求头

- 过于宽松的 CORS（`Access-Control-Allow-Origin: *` 且携带凭证）
- 缺失安全响应头（CSP、X-Frame-Options、X-Content-Type-Options）
- 暴露内部请求头或堆栈信息

## 运行时风险

- 无限循环、递归调用或大型内存缓冲区
- 外部调用缺失超时、重试或速率限制
- 请求路径上的阻塞操作（异步上下文中的同步 I/O）
- 资源耗尽（文件句柄、连接数、内存）
- ReDoS（正则表达式拒绝服务攻击）

## 加密

- 弱算法（将 MD5、SHA1 用于安全目的）
- 硬编码的 IV 或 Salt
- 加密未附带认证（ECB 模式、无 HMAC）
- 密钥长度不足

## 竞态条件

竞态条件是一类隐蔽的缺陷，会导致间歇性故障和安全漏洞。需特别关注以下场景：

### 共享状态访问

- 多线程/协程/异步任务未经同步即访问共享变量
- 全局状态或单例被并发修改
- 延迟初始化未正确加锁（双重检查锁定问题）
- 在并发上下文中使用非线程安全的集合

### Check-Then-Act（TOCTOU）

- `if (exists) then use` 模式未使用原子操作
- `if (authorized) then perform` 但授权状态可能变化
- 文件存在性检查后接文件操作
- 余额检查后接扣款（金融操作）
- 库存检查后接下单

### 数据库并发

- 缺失乐观锁（`version` 列、`updated_at` 检查）
- 缺失悲观锁（`SELECT FOR UPDATE`）
- 读-改-写操作未使用事务隔离
- 计数器递增未使用原子操作（`UPDATE SET count = count + 1`）
- 并发插入导致唯一约束冲突

### 分布式系统

- 共享资源缺失分布式锁
- Leader 选举中的竞态条件
- 缓存失效竞争（写入后读到过期数据）
- 事件排序依赖但缺少正确的顺序保证
- 集群操作中的脑裂场景

### 需标记的常见模式

```
# 危险模式：
if not exists(key):       # TOCTOU
    create(key)

value = get(key)          # 读-改-写
value += 1
set(key, value)

if user.balance >= amount:  # Check-then-act
    user.balance -= amount
```

### 自问

- "如果两个请求同时到达这段代码会怎样？"
- "这个操作是原子的还是可被中断的？"
- "这段代码访问了哪些共享状态？"
- "在高并发下表现如何？"

## 数据完整性

- 缺失事务、部分写入或不一致的状态更新
- 持久化前校验不足（类型隐式转换问题）
- 可重试操作缺失幂等性
- 并发修改导致更新丢失
